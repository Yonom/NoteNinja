## Quickstart

Create a `.env.local` file with the following two API keys:

```
LIVEBLOCKS_SECRET_KEY=
ASSSEMBLYAI_API_KEY=
```

Then run `yarn install` and `yarn dev`.

(Requires NodeJS and Yarn to be installed)
